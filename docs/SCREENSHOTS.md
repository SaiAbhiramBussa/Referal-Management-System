# Flow Builder Screenshots

This folder contains screenshots of the Flow Builder UI.

## How to Capture Screenshots

1. Start the application:
   ```bash
   docker-compose up -d
   ```

2. Open the frontend at http://localhost:3000

3. Use the browser's screenshot or snipping tool to capture:
   - **flow-builder-overview.png**: Main canvas with sample nodes
   - **node-palette.png**: Sidebar showing node types
   - **condition-editor.png**: Condition node configuration
   - **action-editor.png**: Action node configuration
   - **json-preview.png**: JSON export preview
